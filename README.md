# DiaryMe

A privacy-first, local-only diary web app built with React, TailwindCSS, and Dexie.js.

## Features
- Write diary entries with mood, tags, images, and Markdown
- Timeline and calendar views
- Export entries as Word, PDF, or EPUB
- 100% local storage (IndexedDB)
- Responsive, modern UI
- No cloud, no account, no tracking

## Usage
- Run `npm install` to install dependencies
- Run `npm start` to launch the app

## Deployment
- For Netlify or static hosting, `_redirects` file is included for SPA routing

## Support
- Use the Buy Me a Coffee button in the app to support development
